# 多语言支持 (i18n) 功能规格

## 概述

为账本 App 添加多语言支持，默认支持中文和英文。

## 背景

当前 App 仅支持中文，需要扩展支持英文以及其他语言。

## 功能需求

### 1. 语言检测与Fallback
- App 启动时首先读取 localStorage 中的语言设置
- 如果没有保存的语言设置，读取浏览器/系统语言 (navigator.language)
- 如果系统语言不是 'zh' 或 'en'，fallback 为 'en' (英文)

### 2. 语言切换
- 在设置页面添加语言选择下拉框
- 用户选择语言后立即保存到 localStorage
- 语言切换后所有 UI 文本立即更新，无需刷新

### 3. 语言文件
- 中文: zh (Simplified Chinese)
- 英文: en (English)

## 验收标准

1. **默认语言**: 跟随系统语言（zh/en），非支持语言 fallback 为英文
2. **手动设置**: 用户可在设置页面切换语言
3. **持久化**: 语言设置保存在 localStorage，刷新/重启后保持
4. **即时生效**: 切换语言后 UI 立即更新，无需刷新页面

## 实施顺序

1. 安装 i18n 依赖
2. 创建 i18n 配置和语言文件
3. 在 App.tsx 集成 i18n
4. 提取并翻译各页面文本
5. 在设置页面添加语言切换 UI
6. 测试验证
