# 深色模式支持

**Created**: 2026-03-18
**Status**: Draft

## 概述

为账本 App 添加深色模式支持，包括主题切换、持久化和深色模式适配。

## 现状分析

- ✅ Tailwind 配置了 `darkMode: ["class"]`
- ✅ CSS 变量已定义 light/dark 两套
- ✅ 部分滚动条样式已适配深色模式
- ⚠️ 存在硬编码颜色需要修复（如 `bg-red-50`）

## 配色方案（保持不变）

| 用途 | 颜色 |
|------|------|
| Primary | 蓝色 #3B82F6 |
| 收入 | 绿色 #22C55E |
| 支出 | 红色 |

## 功能需求

1. **主题切换**：用户可在设置页面切换浅色/深色/自动模式
2. **持久化**：用户选择保存在 localStorage
3. **自动跟随**：支持跟随系统主题

## 受影响文件

- 新增：`src/contexts/ThemeContext.tsx`
- 修改：`src/App.tsx`
- 修改：`src/pages/SettingsPage.tsx`
- 修复：深色模式下的问题颜色

## 验收标准

1. 设置页面可切换浅色/深色/自动模式
2. 刷新页面保持用户选择的主题
3. 深色模式下所有颜色正确显示
