# 记录编辑和删除功能

**Created**: 2026-03-18
**Status**: PENDING

## 概述

为账本记录添加编辑和删除功能

---

## 当前状态

- 后端 API：已实现 `PUT /records/:id` 和 `DELETE /records/:id` ✅
- 前端 API：已有 `recordApi.update()` 和 `recordApi.delete()` ✅
- 前端 UI：**需要实现**

---

## 用户需求

- 在首页记录列表中，每条记录可以编辑和删除
- 编辑：跳转到编辑页面，修改记录信息
- 删除：弹出确认对话框，确认后删除

---

## 设计方案

### 1. 首页记录卡片修改 (HomePage.tsx)

在每条记录卡片右侧添加操作按钮：
- 编辑按钮（铅笔图标）- 跳转到编辑页面
- 删除按钮（垃圾桶图标）- 点击后弹出确认对话框

**布局**：
```
[分类图标] [分类名称 · 日期 · 备注]     [编辑] [删除]
                [标签]                  ¥金额
```

### 2. 编辑页面 (AddRecordPage.tsx 或新建 EditRecordPage)

复用 AddRecordPage 逻辑：
- 接收记录 ID 作为参数
- 加载记录数据并填充表单
- 提交时调用 `recordApi.update()`
- 成功后返回首页

**URL 设计**：`/edit/:id`

### 3. 删除确认对话框

使用现有 shadcn/ui Dialog 或 AlertDialog：
- 显示确认文案："确定删除这条记录吗？"
- 取消/确认按钮
- 确认后调用 `recordApi.delete()`
- 刷新记录列表

---

## 验收标准

1. 首页每条记录右侧显示编辑和删除按钮
2. 点击编辑按钮跳转到编辑页面
3. 编辑页面可以修改所有字段并保存
4. 点击删除按钮弹出确认对话框
5. 确认删除后记录被删除并刷新列表
6. 构建通过
