# 首页布局优化与统计页年份选择器修复

**Created**: 2026-03-18
**Status**: PENDING

## 概述

修复以下 UI 问题：
1. 首页顶部右侧按钮与底部 tab 冲突 - 移除按钮
2. 统计页面年份选择器深色主题样式不佳 - 改用 shadcn/ui Select 组件

---

## 用户需求

### 问题 1：首页顶部按钮冲突
- **现象**：首页顶部标题栏右侧的统计和设置按钮与底部 tab 导航位置冲突
- **期望**：移除这两个按钮，保持界面整洁

### 问题 2：统计页年份选择器深色样式
- **现象**：统计页面右上角的年份选择器使用原生 `<select>`，深色模式下样式不佳
- **期望**：使用 shadcn/ui Select 组件，深色模式下样式一致

---

## 设计方案

### 方案概述

| 问题 | 方案 | 理由 |
|------|------|------|
| 首页按钮冲突 | 移除顶部右侧按钮 | 简化界面，统计和设置功能可通过其他入口访问 |
| 年份选择器样式 | 使用 Select 组件 | 项目已有 shadcn/ui，保持样式一致性 |

### 详细设计

#### 1. 首页修改 (HomePage.tsx)

**移除内容**：
- 移除 header 中的统计按钮 (`<Link to="/stats">`)
- 移除 header 中的设置按钮和菜单

**保留内容**：
- 保留标题栏左侧的账本名称显示
- 底部 FAB 按钮保持不变
- 底部导航栏保持不变

#### 2. 统计页修改 (StatsPage.tsx)

**替换组件**：
- 将原生 `<select>` 替换为 `@/components/ui/select` 组件
- 确保 year selector 在深色/浅色模式下样式一致

---

3. ✅ 统计页面年份选择器在深色模式下样式正常
