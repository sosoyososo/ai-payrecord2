# 导航修复 - 统一顶部标题栏

**Created**: 2026-03-18
**Status**: Draft

## 问题分析

1. **FAB 仍被遮挡** - bottom-20 仍不够，需要 bottom-24 或更大
2. **保存按钮被遮挡** - 添加记录页面底部保存按钮被挡
3. **二级页面无返回按钮** - 之前的修改可能丢失了

## 解决方案

### 统一标题栏组件设计

每个页面使用统一的顶部标题栏：
- 首页：显示账本名称 + 设置按钮
- 二级页面：显示返回按钮 + 页面标题

### FAB 和底部元素定位

- FAB: `bottom-24` (避免被底部导航遮挡)
- 页面底部padding: `pb-24` (为底部导航留空间)

### 受影响页面

1. **首页 (HomePage)** - 标题栏显示账本名称和设置入口
2. **添加记录 (AddRecordPage)** - 返回按钮 + 标题
3. **统计 (StatsPage)** - 返回按钮 + 标题
4. **预算 (BudgetPage)** - 返回按钮 + 标题
5. **设置 (SettingsPage)** - 返回按钮 + 标题
6. **账本 (LedgerPage)** - 返回按钮 + 标题
7. **分类 (CategoryPage)** - 返回按钮 + 标题
8. **标签 (TagPage)** - 返回按钮 + 标题
9. **导出 (ExportPage)** - 返回按钮 + 标题

## 验收标准

1. 每个页面顶部都有标题栏
2. 二级页面标题栏左侧有返回按钮，点击可返回上一页
3. FAB 完整可见，不被底部导航遮挡
4. 底部保存按钮完整可见
5. PC 端布局正常（使用左侧导航栏时）
